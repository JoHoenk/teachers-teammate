# Third-Party Notices — Installers / Packaging

This file covers third-party software embedded in the **installer artifacts**
produced by the scripts in this directory. It is separate from
`third_party_licenses/` (and the in-app Third-Party Licenses dialog), which
covers the software bundled **inside** the Teacher's Teammate application itself.

## Scope by platform

- **Windows** — `windows.nsi` is compiled with `makensis` (NSIS) to produce
  `dist/teachers-teammate-<version>-w64-setup.exe`. That installer **embeds the
  NSIS self-extractor stub and its standard plugins/compression modules**, so the
  NSIS license applies to the distributed `setup.exe`. See below.
- **macOS** — `make_dmg.sh` wraps the `.app` bundle into a `.dmg` using Apple's
  system `hdiutil`. No third-party installer code is embedded in the artifact.
- **Linux** — `make_deb.sh` builds a `.deb` (an `ar` archive of the binary plus
  metadata) using the system `dpkg-deb`. No third-party installer code is
  embedded. The package declares `Depends: tesseract-ocr`, which the user's
  package manager installs separately — it is not redistributed here.

The application payload these installers wrap is identical on every platform and
is documented in `third_party_licenses/`.

### Compression modules

Depending on the compressor selected at build time, the generated `setup.exe`
may embed one of NSIS's compression modules:

- **zlib** — zlib/libpng license (reproduced above).
- **bzip2** — the bzip2 license (BSD-style; includes an author-attribution
  requirement). Full text: https://github.com/kichik/nsis/blob/master/COPYING
- **LZMA** (NSIS default) — Common Public License version 1.0 (CPL-1.0). NSIS
  grants a special exception permitting static or dynamic linking to the LZMA
  module files without subjecting the linked code to CPL terms; modifications to
  the LZMA module files themselves remain under CPL-1.0. Full CPL-1.0 text and
  the exception: https://github.com/kichik/nsis/blob/master/COPYING

All of the above permit redistribution of the compiled installer. The zlib/libpng
and bzip2 licenses impose no attribution requirement on binary (installer)
redistribution; this notice is provided as the acknowledgment the zlib/libpng
license describes as "appreciated."
